# cvr-jp

ChilltoutVRのメニューで日本語が文字化けしないようにし、またメニュー項目をある程度日本語に翻訳するツールです。

ログイン画面やカメラなどメニュー以外の場所とファイルに記載されてない部分は翻訳されてません。

2025r179 で動作確認しています。それ以外のバージョンでも概ね動作すると思いますが、ツールの更新が必要になるかも知れません。

## 使用方法

https://github.com/Narazaka/chilloutvr-jp-translation-tool/releases/latest から cvr-jp-*.zip をダウンロード

cvr-jp-apply.exeを実行する。

注意: もし「既にパッチが当たっていました」などの文言が出た場合は、SteamライブラリのChilloutVRの管理（歯車メニュー）から「インストール済みファイル」→「ゲームファイルの整合性を確認」を一度実行してから、改めて実行して下さい。

### 「ChilloutVRが見付かりません。Steamのインストール先を確認してください。」

ChilloutVRのインストール先やStreamデフォルトインストール先を変更している場合は、ChilloutVR.exeの入っているフォルダパスをcvrBasePath.txtに書いて上書き保存して下さい。

ChilloutVR.exeの入っているフォルダパスは、SteamライブラリのChilloutVRの管理（歯車メニュー）から「インストール済みファイル」→「参照...」のボタンを押すと立ち上がるExplorerのパス部分をコピーして下さい。

## フォントについて

フリーフォントの「BIZ UDPゴシック」を使用しています。
